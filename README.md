# 🚀 User Management Dashboard

A modern and responsive **User Management Dashboard** built using React.js.
This application fetches user data from a public API and allows users to search, sort, and view detailed information seamlessly.

---

## 📌 Features

- 🔍 Search users by name (case-insensitive)
- 🔤 Sort users (A → Z / Z → A)
- 👤 View detailed user information
- 🔗 Clickable Email, Phone, and Website
- ⚡ Fast performance using React Hooks & useMemo
- 🌐 API integration with error handling
- ⏳ Loading state handling
- 📱 Fully responsive design (Mobile, Tablet, Desktop)

---

## 🛠️ Tech Stack

- **React.js** (Functional Components)
- **React Hooks** (useState, useEffect, useContext, useMemo)
- **React Router**
- **Context API** (State Management)
- **CSS3**

---

## 🌍 API Used

- https://jsonplaceholder.typicode.com/users
